Web Server

1. What is a web server?

    --> A Web server refers to a server software or hardware dedicated to running said software, what can serve contents to the World Wide Web. It also processes incoming network request

2. What does a web server?

    --> It stores, process and delivers the web pages to clients.

3. How can content be stored on a web server?

    --> On the Hypertext Transfer Protocol (HTTP)

4. How is content in webpages described?

    --> With a HTML document

Web Client

1. What is a web client?

    --> It is a client-side component within the Java 2 Platform Enterprise Edition (J2EE) and it's a distributed multi-tiered application model used for building and developing enterprise applications.

2. What does a web client?

    --> It performs client-side operations as they might need access to information available only on the client side, like user input, or because the server lacks the processing power necessary in such operations.

3. Web Client and Web Browser is the same?

    --> The main difference between a Web browser and a Web server is that Web browser requests for the document and services, and act as an interface between a client and a server which displays the web content. On the other hand, Web server accepts, approve, and response to the request made by a web browser for a web document.
